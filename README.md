## Installation
```
Untar: tar -xvf pbviap.zip
```

## Run
```
  ./pbviap_tool.py file
 ```
  

## Results
# Output contains the string:
```
-Result VIAP_STANDARD_OUTPUT_True when the program is safe
```
```
-Result VIAP_STANDARD_OUTPUT_False when a counterexample is found.
```
```
-Result UNKNOWN otherwise.
```
